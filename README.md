# Text Alignment Demo

This is a simple HTML and CSS project that demonstrates different text alignments using CSS. 

## Features

- Left-aligned content
- Center-aligned content
- Right-aligned content
- Card-style sections with hover effect
- Responsive container with max-width
- Smooth hover scaling effect

## Technologies Used

- HTML5
- CSS3 (including box-shadow, transition, and transform)
